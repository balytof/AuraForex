DOWNLOAD AURA V8 INSTITUCIONAL (.ex5)

Copiar o EA para o MetaTrader 5

1. No MT5, clique em Ficheiro → Abrir pasta de dados
2. Navegue até: MQL5 → Experts
3. Cole o ficheiro AuraForex_V8_INSTITUTIONAL.ex5 para esta pasta
4. No MT5, pressione F5 (ou clique direito em Expert Advisors → Actualizar)

2
Autorizar WebRequest (OBRIGATÓRIO)

1. No MT5: Ferramentas → Opções → Especialistas
2. Marque: ☑ "Permitir WebRequest para os seguintes URLs"
3. Clique no "+" e adicione o endereço do nosso servidor:

...

3
Ativar e Aplicar

1. Certifique-se que o botão "Algotrading" no topo do MT5 está VERDE.

3. Na janela que surge, vá ao separador "Inputs".
